# Playground

See how web-meta-scraper works in real time. Enter any URL below to extract its metadata — including Open Graph, Twitter Cards, JSON-LD, oEmbed, and standard meta tags.

The result includes two parts:
- **`metadata`** — The final merged output, resolved by [priority rules](/docs/options#custom-resolve-rules).
- **`sources`** — Raw data from each plugin, so you can see exactly where each value came from.
