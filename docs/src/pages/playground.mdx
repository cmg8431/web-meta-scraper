# Playground

See how web-meta-scraper works in real time. Enter any URL below to extract its metadata — including Open Graph, Twitter Cards, JSON-LD, oEmbed, favicons, RSS/Atom feeds, and robots directives.

The result is displayed in five tabs:
- **Preview** — Visual card showing the OG image, title, description, discovered favicons, feed links, and robots indexing status.
- **Validation** — SEO score (0–100) with categorized issues: errors, warnings, and info.
- **Content** — Extracted main text content with language detection and word count.
- **Metadata** — The final merged output, resolved by [priority rules](/docs/options#custom-resolve-rules).
- **Sources** — Raw data from each plugin, so you can see exactly where each value came from.
