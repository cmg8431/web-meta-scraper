# 플레이그라운드

web-meta-scraper가 실시간으로 어떻게 동작하는지 확인해 보세요. 아래에 URL을 입력하면 Open Graph, Twitter Cards, JSON-LD, oEmbed, 표준 메타 태그를 포함한 메타데이터를 추출합니다.

결과는 두 부분으로 구성됩니다:
- **`metadata`** — [우선순위 규칙](/ko/docs/options#커스텀-리졸브-규칙)에 따라 병합된 최종 결과.
- **`sources`** — 각 플러그인의 원본 데이터로, 각 값이 어디서 왔는지 확인할 수 있습니다.
